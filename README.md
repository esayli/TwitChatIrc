## Getting Started

Install the module with: `npm install twitchatirc`

```js
var twitchatirc = require('twitchatirc');
twitchatirc.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g twitchatirc
$ twitchatirc --help
$ twitchatirc --version
```


